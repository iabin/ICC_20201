Práctica 1

Introducción a las herramientas del laboratorio

Debido a que vamos a usar este repositorio para absolutamente todo, esta práctica es OBLIGATORIA.

Objetivo

El objetivo de esta práctica es familiarizarse con las tecnologías que se usarán en el 100% de las siguientes prácticas, así como familiarizarse con el formato de las entregas de las prácticas y los proyectos.

Al terminar la práctica el alumno tendrá un repositorio personal para el curso de ICC y tendrá una noción general del proceso de programación.

Desarrollo

Antes de comenzar a programar debemos comenzar a aprender a usar el ambiente de desarrollo, y las herramientas necesarias para esta tarea.

Las principales herramientas serán:

  • La consola de tu sistema operativo
  • Git
  • El compilador de Java
  • El intérprete de la JVM
  • Editor de texto de tu preferencia

Primero lo primero

  1. El primer paso de la práctica es crear repositorio público en github con el siguiente nombre obligatoriamente

ICC_20201_NUMCUENTA donde NUMCUENTA es tu número de cuenta.

ejemplo*              ICC_20201_301233514

El repositorio será público, y no debe ser inicializado con ningún README

  1. Crear un documento en LaTex de tipo reporte contestando las siguientes preguntas
  1. ¿Qué es una clase?
  2. ¿Cúal es la diferencia entre una clase y un objeto?
  3. Dar 3 ejemplos de clases y un par de objetos por cada clase
  4. ¿Cúal es el propósito de la documentación?
  5. ¿Qué ventajas tiene hacer el diseño de un programa antes de codificar?
  6. ¿Qué comando se usa para ejecutar un programa en java?
  7. ¿Qué comando se usa para compilar un programa en java?

  1. En la carpeta que hayas elegido para todas tus prácticas y proyectos crea un archivo README.txt, que tendrá tu número de cuenta y tu nombre, dentro de esa carpeta, crea otra carpeta que se llame practica01 y dentro de practica01, ubica el pdf del paso 2

Las carpetas estarán de la siguiente manera.

./

 |── practica01

 |   └── preguntas.pdf

 └── README.txt

  1. Subir toda la carpeta anterior al repositorio creado en github
  2. Una vez que tengas tu repositorio listo, copiar la dirección del repositorio y agregarlo a la siguente lista, y también tu correo de CIENCIAS, puse un ejemplo para que vean como es.

https://docs.google.com/document/d/11KIYcZcMIKmQquDRfJ6iQlE2PjnIiC377LL7wxzadEg/edit?usp=sharing

Es un documento en google drive. Es importante que hagas este último paso, porque de lo contrario no tendré manera de conocer tu repositorio y calificar tus prácticas.

Esto es toda la práctica, tendrás que hacer esto de git para cada práctica, la primera vez parece complicado, pero en cuanto te acostumbres te darás cuenta de lo rápido, práctico y cómodo que es.
