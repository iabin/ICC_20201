Práctica 6

 Arreglos bidimensionales

Objetivo

El objeto de esta práctica es aprender a utilizar arreglos de más dimensiones.

Desarrollo

Descarga la clase Matrix

Deberás implementar  las clase Matrix con los siguientes 8 métodos.

   //Construye una matriz vacía de n por m

    public Matrix(int n, int m)

    //Construye una matriz dado un arreglo

    public Matrix(double[][] array)

    //Función que calcula producto escalar

    public void scalarProduct(double scalar)

    //Función que calcula suma de matrices

    public void add(Matrix m)

    //Función que calcula producto de matrices

    public void matrixProduct(Matrix m)

    //Función que obtiene el elemento i j

    public double getElement(int i, int j)

    //function set element

    public void setElement(int i, int j,double e)

    //Función que calcula el determinante si es nxn

    public int determinant()

    //Te dice si 2 matrices son iguales

    public boolean equals(Object e)

Deberás implementar otra clase con un método main, que se llame MatrixTests

Que tendrá un método por cada método de la clase matriz que se encargará de probarlo e imprimir en la consola si está bien o mal

Al final deberás tener un método main dentro de MatrixTests que llame a cada método de prueba.
